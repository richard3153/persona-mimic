# Persona Mimic - 世界名人人格预设库

## 技能概述

本技能提供1000位世界知名人物的人格预设，涵盖科学家、艺术家、企业家、运动员、作家、政治家、历史人物等多个领域。

## 名人数据库结构

### 领域分类

| 文件 | 领域 | 人数 | 说明 |
|------|------|------|------|
| 01_scientists.json | 科学家与发明家 | 120+ | 物理学家、化学家、生物学家、计算机科学家等 |
| 02_artists.json | 艺术家与音乐家 | 60+ | 画家、雕塑家、音乐家、建筑师等 |

## 使用方法

### 方式一：直接指定角色

```
用户：像爱因斯坦一样思考
Agent：加载 einstein 预设，采用其思维模式和说话风格
```

### 方式二：描述需求

```
用户：我想了解量子物理，用通俗的方式解释
Agent：自动匹配爱因斯坦、玻尔等物理学家风格
```

### 方式三：跨领域组合

```
用户：我想写一篇像乔布斯一样有说服力的商业演讲
Agent：综合乔布斯的演讲风格进行创作
```

## 注意事项

1. **文化敏感性**：部分历史人物可能有争议，请选择中立正面的角度
2. **时序更新**：数据库会持续更新更多人物
3. **组合使用**：可以同时加载多个角色进行对比分析
4. **创意发挥**：预设是起点，Agent会根据场景进行合理发挥

## 扩展建议

如需添加更多人物，可以：
1. 提供人物的维基百科链接或生平简介
2. 描述其核心特质和说话风格
3. 提供其经典语录或行为模式

Agent会自动整理并添加到数据库中。
